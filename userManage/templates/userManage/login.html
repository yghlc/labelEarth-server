This is a crowd-sourcing system for validating retrogressive thaw slumps or thermokarst landslides identified from over 200 TB of ArcticDEM using deep learning. Retrogressive thaw slump is a type of permafrost thawing, which is causing concerns because it changes local environment and affects northern communities, ecosystems, and global carbon feedback.
